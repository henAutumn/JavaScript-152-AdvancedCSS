# Examples

---

```css

body {
  padding: 100px;
  background: #eee;
}

.wrapper {
  text-align: center;
  vertical-align: middle;
  margin-bottom: 100px;
  border: 10px solid black;
  background: white;
  padding-bottom: 100px;
  -webkit-box-shadow: 0px 40px 60px -20px rgba(0, 0, 0, 0.2);
          box-shadow: 0px 40px 60px -20px rgba(0, 0, 0, 0.2);
}

.wrap{
  margin: 10%;
  width: 1000px;
  height: 800px;
  border: 3px solid black;
}

.absolute{
  position: absolute;
  width: 150px;
  height: 100px;
  background: blue;
  border: 4px solid blue;
  top: 0px;
  left: 0px;
}

.relative{
  width: 200px;
  height: 200px;
  position: relative;
  border: 4px solid purple;
  background: purple;
  top: 100px;
  left: 300px;
}

.static{
  width: 200px;
  height: 200px;
  position: static;
  border: 4px solid red;
  background: red;
/*  top & left dont affect  */
}

.inherit{
  width: 200px;
  height: 200px;
  position: inherit;
  border: 4px solid navy;
  background: navy;
  top: 100px;
  right: 100px;
}

.fixed{
  width: 200px;
  height: 200px;
  position: fixed;
  border: 4px solid green;
  background: green;
  top: 400px;
  right: 400px;
}
```
