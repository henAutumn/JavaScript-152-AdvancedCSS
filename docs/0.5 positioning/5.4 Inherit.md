# Inherit

---

#### An element give a position of _inherit_

* _Forces_ element to use positioning of **parent**

---

Let's add another shape to our box.

```html
<div class="circle"></div>
```

```css
.circle {
  width: 200px;
  height: 200px; 
  position: inherit; 
  border-radius: 50%; 
  background: navy; 
  top: 100px; 
  left: 100px;
```
