# Grids

---

There are several ways to set up a grid.

## Grid Tracks

```
box-sizing: border-box;
```

```html
<h2>Grid Tracks</h2>
<div class="wrappert">
  <div class="track">One</div>
  <div>Two</div>
  <div>Three</div>
  <div>Four</div>
  <div>Five</div>
</div>
```

## Grid-Templates

| Property | Value |
| :--- | :--- |
| grid-template-columns |  |
| grid-template-rows |  |
| grid-template-areas |  |

```html
<div class="wrapper">
  <div class="one">One</div>
  <div class="two">Two</div>
  <div class="three">Three</div>
  <div class="four">Four</div>
  <div class="five">Five</div>
  <div class="six">Six</div>
</div>
```

```css
* {box-sizing: border-box;}

.wrapper {
  max-width: 940px;
  margin: 0 auto;
}
```

```css
.wrapper > div {
  border: 2px solid rgba(0,128,128,1.0);
  border-radius: 5px;

}
.wrapper {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 10px;
  grid-auto-rows: minmax(100px, auto);
}
.one {
  grid-column: 1 / 3;
  grid-row: 1;
}
.two { 
  grid-column: 2 / 4;
  grid-row: 1 / 3;
}
.three {
  grid-column: 1;
  grid-row: 2 / 5;
}
.four {
  grid-column: 3;
  grid-row: 3;
}
.five {
  grid-column: 2;
  grid-row: 4;
}
.six {
  grid-column: 3;
  grid-row: 4;
}
```

.wrapper {
